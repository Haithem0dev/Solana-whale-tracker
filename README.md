# Solana Whale Tracker 🐋

This is a lightweight tool I built to help monitor whale activity on Solana. It's designed to be simple, efficient, and easy to integrate with Telegram bots later.

### Features
* **Real-time Monitoring:** Tracks high-value transactions using Helius RPC.
* **Developer Friendly:** Clean Python code, easy to customize.
* **AI-Augmented:** Built with a focus on rapid iteration and modern engineering practices.

### How to use
1. Clone the repository.
2. Add your Helius API Key.
3. Run `python whale_tracker.py`.
